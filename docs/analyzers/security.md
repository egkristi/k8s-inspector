# Security Compliance

The Security Analyzer evaluates your clusters against industry-standard benchmarks to ensure a hardened posture.

## Features
* **CIS Benchmarks:** Checks against the Center for Internet Security (CIS) Kubernetes benchmarks.
* **Pod Security Standards (PSS):** Identifies privileged containers and containers running as root.
* **RBAC Audits:** Highlights over-permissive ClusterRoles (e.g., wildcard permissions).
* **Network Policies:** Ensures namespaces implement default deny-all policies.
