# Welcome to k8s-inspector 2.0 🐦‍⬛

> **Not just monitoring — *understanding*. Not just alerts — *answers*. Not just problems — *solutions*.**

k8s-inspector 2.0 is an intelligent, AI-powered observability platform for Kubernetes. It goes beyond traditional metrics by providing root cause analysis, cost optimization, security compliance checks, and automated remediation.

## Why k8s-inspector?

Traditional tools tell you *what* went wrong (e.g., "Pod restarted 5 times"). 
k8s-inspector tells you *why* it happened and *how* to fix it (e.g., "OOMKilled because memory limit is too low. Click here to increase limit to 768Mi").

### Key Capabilities

* **🧠 AI-Powered Root Cause Analysis:** Explains failures in plain English.
* **💰 Intelligent Cost Optimization:** Detects orphaned PVCs, unused LoadBalancers, and rightsizes deployments.
* **🔮 Predictive Failure Detection:** Uses ML to predict when resources will exhaust.
* **🔐 Security Compliance:** Checks against CIS benchmarks and NSA/CISA guidelines.
* **🤖 Auto-Remediation Playbooks:** Fixes issues automatically via YAML-based playbooks.
